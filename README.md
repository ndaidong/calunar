# yearly-calendar

Util for calculating lunar date based on Ho Ngoc Duc implementation.

Refer: http://www.informatik.uni-leipzig.de/~duc/amlich/

## Usage

```
LunarConvertor.solar2Lunar(9, 2, 2016, 7); // return [2, 1, 2016, 0]
```
